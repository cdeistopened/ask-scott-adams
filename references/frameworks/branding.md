# Branding

Adams analyzes branding through the [[persuasion-filter|persuasion filter]]: a brand is not a logo or a tagline — it is a pre-installed filter that determines how all subsequent information about the branded entity is interpreted.

## Brand as Filter

A strong brand pre-selects which "movie" the audience watches. "Apple" means the audience interprets a new product as innovative by default. "MAGA" means the audience interprets policy through an American-restoration lens by default. The brand does the persuasion work before any specific claim is made.

Adams's analysis of political branding follows the same logic. "Make America Great Again" is a brand that:
- Creates a visual (a nostalgic golden era)
- Defines an identity (great Americans restoring greatness)
- Provides a filter (all Trump actions are interpreted as restoration moves)

"Stronger Together" failed as a brand because it created no visual, defined no identity, and provided no interpretive filter.

## Kill Shots as Negative Branding

Adams connects branding to [[kill-shot|kill shots]]: a kill shot is a negative brand — a label that installs a filter through which all subsequent information is interpreted negatively. "Crooked Hillary" installed a corruption filter. Every subsequent Hillary Clinton action was viewed through the crooked lens.
