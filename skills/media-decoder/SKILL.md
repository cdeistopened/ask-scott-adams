# Media Decoder

Analyze any news story, public controversy, or media narrative through Scott Adams' analytical toolkit. Adams spent years on Coffee with Scott Adams doing real-time media analysis, identifying the persuasion techniques, cognitive biases, and narrative structures that most consumers miss.

**Critical note:** This is analytical, not political. Adams's material is politically charged, but this skill analyzes persuasion TECHNIQUE, not political POSITIONS. Present both movies. Identify the persuasion on all sides. Let the user decide which movie they find more predictive.

## When to Use

The user wants to understand a news story, controversy, or public debate at a deeper level than the surface narrative. Or they're confused by how two groups can look at the same events and reach opposite conclusions.

## How to Run This

Ask: "What's the story, controversy, or narrative you want decoded? Share a link, headline, or summary."

Then run through the analysis framework.

---

## The Analysis Framework

### Step 1: The Two Movies

Adams's foundational media analysis tool: identify the two (or more) internally consistent movies people are watching about the same set of facts.

**Present both movies side by side:**

| Dimension | Movie A | Movie B |
|-----------|---------|---------|
| What happened | [Movie A's version] | [Movie B's version] |
| Why it happened | [Movie A's explanation] | [Movie B's explanation] |
| What it means | [Movie A's interpretation] | [Movie B's interpretation] |
| What comes next | [Movie A's prediction] | [Movie B's prediction] |
| Who is the villain | [Movie A's villain] | [Movie B's villain] |
| Who is the hero | [Movie A's hero] | [Movie B's hero] |

Adams's key insight: "You can tell which movie someone is watching by asking what they predict will happen next. People in different movies make different predictions. The movie that predicts better is the one closer to reality — or at least the one you should weight more heavily."

**Important:** Present both movies with equal analytical rigor. Do not strawman either side. If you cannot make Movie B sound compelling and internally consistent, you are probably in Movie A's cognitive dissonance.

### Step 2: Persuasion Technique Identification

For each side of the narrative, identify the persuasion techniques being used:

**Common techniques Adams identifies in media:**
- **Kill shot labels:** Has either side branded the other with a sticky label?
- **Visual persuasion:** Which side has the more vivid mental image?
- **Pacing and leading:** Is either side first agreeing with you before redirecting?
- **High-ground maneuver:** Is either side claiming an unassailable moral position?
- **Anchoring:** Has either side set an extreme position to make a moderate position feel like a concession?
- **Social proof:** Is either side claiming "everyone agrees" or "experts say"?
- **Identity play:** Is either side framing the issue as "people like us believe X"?
- **Fear sell:** Is either side using disproportionate threat language?

Present the persuasion audit for BOTH sides:

```
MOVIE A Persuasion Techniques:
- [Technique]: [How it's being used]
- [Technique]: [How it's being used]

MOVIE B Persuasion Techniques:
- [Technique]: [How it's being used]
- [Technique]: [How it's being used]
```

### Step 3: Hoax Funnel Test

Adams's "hoax funnel" describes how sensational claims evolve: they start huge and scary, then narrow as facts emerge, until the original claim is either confirmed or revealed as essentially nothing.

**Apply the test:**
1. **What was the original claim?** (The sensational version that first got attention)
2. **What do we actually know now?** (The verified facts, stripped of interpretation)
3. **How much has the claim narrowed from the original?**
4. **Is the media still using the original framing despite the narrowing?**

Adams's diagnostic: "If the claim has narrowed by 90% but the coverage intensity hasn't changed, you're probably watching a hoax funnel in action."

**Note:** Not every story is a hoax funnel. Some stories start big and stay big because the original claim was accurate. The test is whether the claim has narrowed while the rhetoric hasn't.

### Step 4: Cognitive Dissonance Check

Adams has a specific set of "tells" for identifying cognitive dissonance in media coverage and public reaction (from Episode 81 and many others):

- **One-variable explanation:** "It's just because they're [stupid/racist/corrupt/crazy]." Real analysis uses multiple variables.
- **Absurd absolute expansion:** Taking a reasonable point and expanding it to an absurd extreme ("So you're saying we should let children drive cars?")
- **Mind-reading:** "What they REALLY mean is..." or "The real reason is..." — attributing secret motives without evidence.
- **Word salad:** Long, complex responses that use many words but don't actually say anything falsifiable.
- **Unspecific doom forecast:** "This could tear apart the fabric of civilization." What does that look like specifically?
- **Ignoring prediction track record:** The person's past predictions have been wrong, but they speak with the same confidence.
- **"So..." tell:** Sentences starting with "So you're saying..." almost always follow with a strawman.

Flag any of these tells you observe in the coverage or public reaction.

### Step 5: Prediction Test

**Ask for each movie:**
- What does this movie predict will happen next?
- What would prove this movie wrong?
- Has this movie's past predictions about similar situations been accurate?

Adams's ultimate arbiter: "I can't tell you who's right. I can tell you which filter on the world has the best track record of predicting what happens next."

## Delivering the Analysis

```
STORY: [One-sentence summary]

MOVIE A: [Name/description]
[2-3 sentence summary of this interpretation]
Persuasion techniques: [list]
Predicts: [what happens next]

MOVIE B: [Name/description]
[2-3 sentence summary of this interpretation]
Persuasion techniques: [list]
Predicts: [what happens next]

HOAX FUNNEL STATUS: [Not applicable / Early stage / Narrowing / Confirmed substance]

COGNITIVE DISSONANCE TELLS: [Any tells observed in coverage or reaction]

WHICH MOVIE PREDICTS BETTER: [Based on past track records of similar claims]
```

## What This Skill Does NOT Do

- It does not tell the user which side is politically correct
- It does not endorse Adams's personal political views
- It does not claim objectivity — it claims analytical rigor using a specific framework
- It presents both movies and lets the user decide which one they find more predictive
